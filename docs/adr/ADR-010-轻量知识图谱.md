# ADR-010: 轻量知识图谱（Phase 3）

**状态**：已采纳  
**日期**：2026-05-26

## 决策

- 参考 LightRAG「实体-关系-Chunk 关联」，**不上 Neo4j**
- SQLite 表 `kg_relations` 存三元组；NetworkX 内存图 + LRU 缓存
- 入库时异步抽取（LLM / Mock 规则），与 FTS 同步 hook 并列
- `relational` 路由：`GraphRetriever` 1–2 hop → chunk，与 `HybridRetriever` RRF 融合
- API `GET /api/knowledge-bases/{kb_id}/graph`；前端知识库详情「知识图谱」Tab

## 配置

| 变量 | 默认 | 说明 |
|------|------|------|
| `GRAPH_ENABLED` | true | 开关 |
| `GRAPH_MAX_HOPS` | 2 | 多跳深度 |
| `GRAPH_MAX_TRIPLES_PER_CHUNK` | 5 | 每 chunk 上限 |
| `RETRIEVAL_ABSTAIN_ENABLED` | true | 低置信空检索（负例抑制） |
| `RETRIEVAL_ABSTAIN_MIN_SCORE` | 0.20 | abstention 分数阈值 |
| `RETRIEVAL_ABSTAIN_MIN_OVERLAP` | 0.10 | abstention 词重叠阈值 |
