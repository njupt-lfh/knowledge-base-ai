# ADR-002: 反馈与质量分（Phase 1.2）

**状态**：已采纳  
**日期**：2026-05-26

## 决策

- 表 `chunk_quality`：`quality_score`、`like/dislike/correction_count`、`needs_review`
- 公式：`0.4×norm(hit) + 0.3×like_rate - 0.2×correction_rate + 0.1×freshness`
- 检索融合：`final = 0.7×retrieval_score + 0.3×quality_score`
- 反馈写入后 `QualityService.apply_feedback` 更新关联 chunk
- API：`GET .../quality/low-quality`，`POST .../recalculate`
