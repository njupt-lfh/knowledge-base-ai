# ADR-004: 入库质量门禁（Phase 1.4）

**状态**：已采纳  
**日期**：2026-05-24

## 决策

- 每个待入库分块：Chroma top-3 相似检索 → 分层阈值
  - L2 &lt; 0.4（≈ cos &gt; 0.92）：**重复**，跳过入库
  - 0.4 ≤ L2 &lt; √0.5（≈ 0.75 ≤ cos &lt; 0.92）：**最多 3 次 LLM** 语义矛盾检测
  - 有矛盾：写入 `knowledge_conflicts`，待人工裁决，不入库
- 文档处理 `_process_document` / `_process_manual` 统一走 `ingest_text_chunks`
- 文档记录 `ingest_duplicate_count`、`ingest_conflict_count`
- API：预检 `POST .../ingestion/precheck`；冲突列表与裁决 `GET/POST .../conflicts`
- `chunk_versions` 延后至后续迭代
