**状态**：已采纳

## 1. 调研

- **RAGAS**：无参考答案亦可评估 faithfulness / answer_relevancy / context_precision / context_recall
- **数据**：使用项目自有知识库 chunk 构建 QA，不引入外部公开数据集（领域不一致）
- **基线**：当前 Pipeline RAG（Chroma + LIKE 混合检索 + 豆包生成）
