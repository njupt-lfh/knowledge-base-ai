# ADR-000: Phase 0 评测体系

**状态**：已采纳（v1.1 演进中）  
**日期**：2026-05-26（更新 2026-06-04）  
**关联**：Phase 0 · [EVAL.md](../EVAL.md) · [RAG_IMPROVEMENT_ROADMAP.md](../RAG_IMPROVEMENT_ROADMAP.md)

## 1. 调研

- **RAGAS**：无参考答案亦可评估 faithfulness / answer_relevancy / context_precision / context_recall
- **数据**：使用项目自有知识库 chunk 构建 QA，不引入外部公开数据集（领域不一致）
- **基线**：Hybrid + Agentic-lite + 火山引擎生成

## 4. 决策

- 保留 RAGAS 四指标 + chunk 级 IR 指标（MRR、NDCG@5、P@k）
- v1 作回归门禁；v2 作能力拉伸与 multi_hop 专项
- 改善路线与验收口径以 [RAG_IMPROVEMENT_ROADMAP.md](../RAG_IMPROVEMENT_ROADMAP.md) 为准
