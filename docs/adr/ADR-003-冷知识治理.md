# ADR-003: 冷知识治理（Phase 1.3）

**状态**：已采纳  
**日期**：2026-05-24

## 决策

- `GovernanceService.scan_suggestions` 聚合：冷知识（90 天 0 命中）、质量分异常、向量近重复（top-3，L2&lt;0.4）
- 建议类型：`duplicate` | `cold_stale` | `high_quality_zero_hit` | `low_quality` | `archive_candidate`
- 可执行动作：`archive` / `deactivate`（禁用 chunk + Chroma 同步）、`boost_faq`（质量分 +0.15）
- API：`GET /api/knowledge-bases/{kb_id}/governance/suggestions`，`POST .../actions`
- 前端：知识库详情「治理建议」Tab；冷知识 Badge 点击跳转该 Tab
