# ADR-005: 对话提炼升级（Phase 1.5）

**状态**：已采纳  
**日期**：2026-05-24

## 决策

- `ConversationExtractService`：LLM 输出 `title/content/tags/entities/gap_type/source_ref`
- **无 `source_ref` 或引用不在对话中 → 拒绝提炼**（防编造）
- 对话结束 `process_after_chat`：自动分类 + 可入库类型结构化入队
- `KNOWLEDGE_ABSENT`：仅 `manual_required`，**不生成** `suggested_content`
- 仅 `USER_PROVIDED` / `USER_CORRECTION` 可走 `POST .../gaps/{id}/ingest`（经 Phase 1.4 门禁）
- `KNOWLEDGE_ABSENT` 入库必须 `manual_content`（人工正文）
- 手动「提炼知识」→ 创建 Gap → 确认时调用 ingest API
